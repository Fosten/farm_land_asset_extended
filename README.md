# farmOS Land Asset Extended

This module is an add-on for the [farmOS](http://drupal.org/project/farm)
distribution.

## Features

Provides additional farm_land_type_field_allowed_values to Land Assets:
- coldframe
- flowerbed
- garden
- orchard
- parking
- path
- road
